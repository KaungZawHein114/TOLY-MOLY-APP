# Task တင်နည်း

Task တင်ရန်

၁။ Dashboard မှ "Post a Task" ကို နှိပ်ပါ။

၂။ အလုပ်အမျိုးအစားကို ရွေးချယ်ပါ။

၃။ အလုပ်ခေါင်းစဉ် ထည့်ပါ။

၄။ အသေးစိတ်ဖော်ပြချက် ရေးပါ။

၅။ ဘတ်ဂျက် သတ်မှတ်ပါ။

၆။ လိုအပ်ပါက ဓာတ်ပုံတင်နိုင်ပါသည်။

၇။ တည်နေရာကို ရွေးချယ်ပါ။

၈။ Preview ကို စစ်ဆေးပြီး Submit လုပ်ပါ။

Task တင်ပြီးနောက် Tasker များက ကြည့်ရှုနိုင်ပါသည်။
